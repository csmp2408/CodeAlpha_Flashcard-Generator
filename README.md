# Flashcard Quiz App

A modern and interactive Flashcard Quiz Application developed using HTML, CSS, JavaScript, Node.js, and Express.js.

The application helps users learn and revise topics using customizable flashcards. Users can create, edit, delete, search, and organize flashcards into categories while tracking their study progress through an intuitive interface.

## Features

- View flashcards one at a time
- Show/Hide answers
- Next and Previous navigation
- Add new flashcards
- Edit existing flashcards
- Delete flashcards
- Search flashcards
- Category-based organization
- Flashcard counter
- Local storage persistence
- Dark mode support
- Responsive design
- Modern user interface

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Node.js
- Express.js
- Local Storage

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the application

```bash
npm start
```

4. Open in browser

```text
http://localhost:3000
```
